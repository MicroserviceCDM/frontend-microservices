# Car Connect
Car Connect is a website that sells cars. Website is written on react.js.

# Technology Stack

- React.js
- Tailwind CSS
- MUI
- react-chat-engine
- stompjs
- vite

# Project Demo

# Installation and Setup

```bash
# Setup the web app 
git clone github.com/MicroserviceCDM/frontend-microservices
cd frontend-microservices
npm install

# Start the web app 
npm run dev
```

For docker
```bash
# Setup the web app 
git clone github.com/MicroserviceCDM/frontend-microservices
cd frontend-microservices
docker compose up
```
